BNP Paribas - Plan Easyfuture

A investments simulator

App

Tech used

Front
  • Angular JS
  • Javascript + Coffee
  • Atom Angular Boilerplate
Back
  • Symfony 2.3 REST API Edition
  • MySql database

Functionalities

A user can add, remove, edit any stock and stock description. A built-in calendar allows one to to add a meeting, eventually linked to stocks. This adds an alert to complete a meeting review associated with every stock linked. Any stock has entries showing the trend of the stock price over time, and an alert shows up when a big fund movement is made.

Complete administration pannel, enabling stock, user and sectors edition, addition or deletion.
